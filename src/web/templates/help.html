Como usar o Lead Finder

Guia completo para encontrar e gerir leads de negocios

O que e o Lead Finder?

O Lead Finder e uma ferramenta de prospeccao que usa a API do Google Maps para encontrar negocios locais. Ideal para agencias de marketing, web designers, e profissionais de vendas B2B que querem encontrar potenciais clientes que precisam dos seus servicos.

Exemplo: Procura por "restaurantes em Lisboa" e encontra todos os restaurantes, com informacoes como rating, reviews, website, e contactos. Depois podes filtrar por aqueles que NAO tem website - esses sao os teus potenciais clientes!

Fluxo de Trabalho

1

Pesquisar

Usa a pagina de Pesquisa para encontrar negocios. Podes pesquisar por tipo de negocio e localizacao.

2

Analisar

Ve os leads na lista, filtra por score, website, ou status. Cada lead tem um score que indica o potencial.

3

Enriquecer

Usa o Enrichment para extrair emails, redes sociais e contactos dos websites dos leads.

4

Contactar

Exporta os leads para CSV/Excel ou sincroniza com o Notion para gerir o teu pipeline de vendas.

Enrichment - Extrair Contactos

O sistema de Enrichment visita os websites dos leads e extrai automaticamente:

  • Emails - Todos os emails encontrados na pagina
  • Redes Sociais - Links para LinkedIn, Facebook, Instagram, Twitter
  • Decisores - Nomes e cargos de pessoas (CEO, Manager, etc.)

Nota: O enrichment so funciona para leads que tem website. Podes enriquecer um lead individual ou varios de uma vez na pagina de Enrichment.

Integracao com Notion

Podes sincronizar os teus leads com uma base de dados do Notion para gerir o teu CRM:

  1. Vai a Settings
  2. Cria uma Integracao Notion e copia o token
  3. Cola o token e testa a conexao
  4. Seleciona a base de dados onde queres guardar os leads
  5. Agora podes sincronizar leads individualmente ou em batch!

Dica: Cria uma base de dados no Notion com colunas como: Nome, Telefone, Email, Website, Score, Status. O sistema vai mapear automaticamente os campos.

Pesquisas Automaticas

Configura pesquisas que correm automaticamente para encontrar novos negocios:

  1. Vai a Automacao
  2. Clica em "Nova Pesquisa Automatica"
  3. Define a query (ex: "dentistas Porto") e o intervalo (ex: a cada 24h)
  4. Ativa a pesquisa e deixa o sistema trabalhar!

Resultado: Recebes notificacoes quando novos leads sao encontrados. O sistema guarda apenas leads novos, sem duplicados.
